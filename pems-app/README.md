# P.E.M.S. — Property Engagement & Management System

P.E.M.S. is an ultramodern cross-platform property management system by ASTEM Software Labs.

## Main Purpose

The system helps land owners and property managers track:

- Properties
- Buildings
- Rooms
- Tenants
- Rented rooms
- Empty rooms
- Rent due dates
- Payments
- Overdue rent
- Maintenance issues
- Reports

## Platforms

- Android
- iOS
- Web

## Technology Stack

- JavaScript
- React
- React Native with Expo
- NativeWind
- TailwindCSS
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- Firebase Hosting

## Core Users

- Super Admin
- Landlord
- Property Manager
- Tenant

## First Version Goal

The first version will allow landlords to:

- Register and login
- Add properties
- Add buildings
- Add rooms
- Assign tenants
- Track occupied rooms
- Track empty rooms
- Record rent payments
- View overdue rent
- View dashboard reports
