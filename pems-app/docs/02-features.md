# Features

## Authentication

- Register
- Login
- Logout
- Forgot password
- Role-based access

## Dashboard

- Total properties
- Total buildings
- Total rooms
- Occupied rooms
- Empty rooms
- Total tenants
- Expected rent
- Rent collected
- Outstanding balance
- Overdue tenants

## Property Management

- Add property
- Edit property
- Delete property
- View property details

## Building Management

- Add building
- Edit building
- Delete building
- View rooms under each building

## Room Management

- Add room
- Edit room
- Mark room as occupied
- Mark room as empty
- Mark room under maintenance

## Tenant Management

- Add tenant
- Assign tenant to room
- View tenant profile
- Remove tenant
- Track tenant rent

- Record rent payment
- View payment history
- Track balance
- Track next due date
- Show overdue payments

## Maintenance

- Tenants can report issues
- Landlords can update issue status

## Notifications

- Rent due reminders
- Payment confirmations
- Overdue alerts
