# Project Overview

## Product Name

P.E.M.S. — Property Engagement & Management System

## Company

ASTEM Software Labs

## Description

P.E.M.S. is a powerful real estate management platform designed for landlords, property owners, and property managers.

It helps users manage properties, tenants, rooms, rent payments, and reports from one modern dashboard.

## Problem Statement

Many landlords manage rent, rooms, tenants, and payments manually using notebooks, WhatsApp messages, or spreadsheets.

This causes:

- Missed rent payments
- Poor tenant records
- Confusion about empty and occupied rooms
- No clear income report
- Difficulty tracking overdue payments
- Poor communication with tenants

## Solution

P.E.M.S. provides a digital system that helps property owners manage all property operations in one place.

## Target Users

- Individual landlords
- Estate owners
- Real estate companies
- Property managers
- Hostel owners
- Apartment managers
- Commercial building managers
