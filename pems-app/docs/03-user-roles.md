# User Roles and Permissions

## Super Admin

The Super Admin controls the whole system.

### Permissions

- Manage all users
- View all landlords
- View all properties
- Suspend accounts
- View system reports
- Manage app settings

## Landlord

The Landlord owns and manages properties.

### Permissions

- Add properties
- Add buildings
- Add rooms
- Add tenants
- Assign tenants to rooms
- Record rent payments
- Track occupied rooms
- Track empty rooms
- View overdue rent
- View reports

## Property Manager

The Property Manager helps the landlord manage daily property activities.

### Permissions

- View assigned properties
- Add rooms
- Add tenants
- Record payments
- Update maintenance issues
- View reports for assigned properties

## Tenant

The Tenant rents a room or property unit.

### Permissions

- View rent status
- View payment history
- Receive rent reminders
- Submit maintenance requests
- Update profile
